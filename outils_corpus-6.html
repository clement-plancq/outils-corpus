usage

  • si vous voulez utiliser Spacy prenez le temps de lire la documentation, ici ce ne sera qu’un coup d’œil incomplet
  • un modèle est une instance de la classe Language, il est adapté à une langue en particulier
  • un modèle incorpore un vocabulaire, des poids, des vecteurs de mots, une configuration
import spacy
nlp = spacy.load('fr_core_news_md')

usage

  • le traitement fonctionne avec un pipeline pour convertir un texte en objet Doc (texte annoté)
  • par défaut tokenizer > tagger > parser > ner > …
  • l’utilisateur peut ajouter des étapes ou en retrancher
nlp = spacy.load('fr_core_news_md', disable=["parser", "ner"])

usage

  • Un objet Doc est une séquence d’objets Token
  • Le texte d’origine est découpé en phrases, tokenizé, annoté en POS, lemme, syntaxe (dépendance) et en entités nommées (NER)
doc = nlp("Dans les derniers jours de mai 1793, un des bataillons parisiens amenés en Bretagne par Santerre fouillait le redoutable bois de la Saudraie en Astillé. On n'était pas plus de trois cents, car le bataillon était décimé par cette rude guerre.")
for token in doc:
    print(token.text, token.tag_, token.lemma_, token.ent_type_)

pour traiter plusieurs textes, utiliser nlp.pipe

docs = list(nlp.pipe(texts))

re-tokenization

  • voir https://spacy.io/usage/linguistic-features#retokenization
doc = nlp("Tu parles sur moi, y'a R")
print([(tok.text, tok.tag_) for tok in doc])
# pb avec tokenization de "y'a"
# ("y'a", 'ADP', "y'a"), ('R', 'NOUN', 'r')

with doc.retokenize() as retokenizer:       
    heads = [(doc[5], 1), (doc[5], 2)]
    retokenizer.split(doc[5], ["y'", "a"], heads=heads)

print([(tok.text, tok.tag_) for tok in doc])

Attention ici c’est l’objet doc qui est modifié, pas le modèle

modification de la tokenization

from spacy.symbols import ORTH, LEMMA, POS, TAG

doc = nlp("Tu parles sur moi, y'a R")
print([(tok.text, tok.tag_, tok.lemma_) for tok in doc])
# pb avec tokenization de "y'a"
# ("y'a", 'ADP', "y'a"), ('R', 'NOUN', 'r')

special_case = [{ORTH: "y'", LEMMA: "y", POS: "PRO"}, {ORTH: u"a"}]
nlp.tokenizer.add_special_case(u"y'a", special_case)
doc = nlp("Tu parles sur moi, y'a R")
print([(tok.text, tok.tag_, tok.lemma_) for tok in doc])
#("y'", 'PRON', 'y'), ('a', 'AUX', 'avoir')

Règles pour entités nommées

from spacy.pipeline import EntityRuler
doc = nlp("Mardi 19 mars, la firme de Mountain View a levé le voile sur Stadia, son service de jeu vidéo à la demande.")
print([(ent.text, ent.label_) for ent in doc.ents])
# ('Moutain View', MISC), ('Stadia', LOC)
ruler = EntityRuler(nlp, overwrite_ents=True)
patterns = [{"label": "LOC", "pattern": "Mountain View"},
    {"label":"MISC", "pattern":"Stadia"}]
ruler.add_patterns(patterns)
nlp.add_pipe(ruler)
doc = nlp("Mardi 19 mars, la firme de Mountain View a levé le voile sur Stadia, son service de jeu vidéo à la demande.")
print([(ent.text, ent.label_) for ent in doc.ents])

Update du modèle de NER

Voir exemple fourni par Spacy

from spacy.util import minibatch, compounding
import random
# training data
TRAIN_DATA = [
    ("Stadia fonctionne grâce à des data centers", {"entities": [(0, 6, "MISC")]}),
    ("la firme de Mountain View a levé le voile sur Stadia", {"entities": [(12, 25, "LOC"), (46, 52, "MISC")]}),
]

ner = nlp.get_pipe("ner")
# add labels
for _, annotations in TRAIN_DATA:
    for ent in annotations.get("entities"):
        ner.add_label(ent[2])

# get names of other pipes to disable them during training
other_pipes = [pipe for pipe in nlp.pipe_names if pipe != "ner"]
with nlp.disable_pipes(*other_pipes):  # only train NER
    for itn in range(100):
        random.shuffle(TRAIN_DATA)
        losses = {}
        # batch up the examples using spaCy's minibatch
        batches = minibatch(TRAIN_DATA, size=compounding(4.0, 32.0, 1.001))
        for batch in batches:
            texts, annotations = zip(*batch)
            nlp.update(
                texts,  # batch of texts
                annotations,  # batch of annotations
                drop=0.5,  # dropout - make it harder to memorise data
                losses=losses,
            )
        print("Losses", losses)

# test the trained model
for text, _ in TRAIN_DATA:
    doc = nlp(text)
    print("Entities", [(ent.text, ent.label_) for ent in doc.ents])
    print("Tokens", [(t.text, t.ent_type_, t.ent_iob) for t in doc])

# save model to output directory
if output_dir is not None:
    output_dir = Path(output_dir)
    if not output_dir.exists():
        output_dir.mkdir()
    nlp.to_disk(output_dir)
    print("Saved model to", output_dir)

    # test the saved model
    print("Loading from", output_dir)
    nlp2 = spacy.load(output_dir)
    for text, _ in TRAIN_DATA:
        doc = nlp2(text)
        print("Entities", [(ent.text, ent.label_) for ent in doc.ents])
        print("Tokens", [(t.text, t.ent_type_, t.ent_iob) for t in doc])
